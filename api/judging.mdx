The Judging system includes routers for managing projects, tables, tracks, judging, and time slots.

## Project Router

### uploadProjects

**Type:** Mutation  
**Authentication:** Protected (ADMIN role required)

Bulk upload projects for judging.

<ParamField path="projects" type="array" required>
  Array of project objects
</ParamField>

<ParamField path="projects[].name" type="string" required>
  Project name
</ParamField>

<ParamField path="projects[].description" type="string" optional>
  Project description (defaults to empty string)
</ParamField>

<ParamField path="projects[].link" type="string" optional>
  Project link/URL (defaults to empty string)
</ParamField>

<ParamField path="projects[].tracks" type="string[]" required>
  Track names this project belongs to
</ParamField>

<ParamField path="projects[].status" type="string" optional>
  Project status (defaults to "Draft")
</ParamField>

**Example:**
```typescript
await trpc.project.uploadProjects.mutate([
  {
    name: "Cool App",
    description: "An amazing project",
    link: "https://github.com/user/cool-app",
    tracks: ["General", "Web Development"],
    status: "Active"
  }
]);
```

## Judging Router

### createJudgingResult

**Type:** Mutation  
**Authentication:** Protected (ADMIN or JUDGE role required)

Submit or update judging scores for a project.

<ParamField path="projectId" type="string" required>
  Project ID
</ParamField>

<ParamField path="tableId" type="string" required>
  Table ID
</ParamField>

<ParamField path="responses" type="array" required>
  Array of rubric responses
</ParamField>

<ParamField path="responses[].questionId" type="string" required>
  Rubric question ID
</ParamField>

<ParamField path="responses[].score" type="number" required>
  Score (minimum 0)
</ParamField>

**Example:**
```typescript
await trpc.judging.createJudgingResult.mutate({
  projectId: "proj123",
  tableId: "table456",
  responses: [
    { questionId: "q1", score: 8 },
    { questionId: "q2", score: 9 }
  ]
});
```

### createTimeSlots

**Type:** Mutation  
**Authentication:** Protected (ADMIN role required)

Generate time slots for all tables and projects.

<ParamField path="slotDurationMinutes" type="number" required>
  Duration of each slot in minutes (minimum 1, default 10)
</ParamField>

<ParamField path="startTime" type="datetime" required>
  Start time for judging (ISO 8601 format)
</ParamField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="endTime" type="datetime">
  Calculated end time of judging
</ResponseField>

<ResponseField name="numTables" type="number">
  Number of tables scheduled
</ResponseField>

**Example:**
```typescript
const result = await trpc.timeSlot.createTimeSlots.mutate({
  slotDurationMinutes: 10,
  startTime: "2024-01-20T10:00:00Z"
});
```

### getAssignmentsAtTime

**Type:** Query  
**Authentication:** Protected (ADMIN role required)

Get table-to-project assignments at a specific time.

<ParamField path="time" type="string" required>
  Time in ISO format
</ParamField>

<ResponseField name="[tableId]" type="object">
  Object mapping table IDs to project info
</ResponseField>

**Example:**
```typescript
const assignments = await trpc.timeSlot.getAssignmentsAtTime.useQuery({
  time: "2024-01-20T10:00:00Z"
});
// Returns: { "table1": { id: "proj1", name: "Cool App" }, ... }
```
